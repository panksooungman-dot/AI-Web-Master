## 13. Index Maintenance Rule

- 신규 문서를 생성하면 해당 폴더 섹션에 행을 추가한다.
- 문서 상태가 바뀌면(Draft → Active, Active → Archived 등) 이 표를 갱신한다.
- 문서를 이동·삭제하는 경우 이 인덱스도 함께 갱신한다.
