# templates

재사용 가능한 **보일러플레이트·스캐폴딩 템플릿**을 보관합니다.

## 포함할 내용

- 페이지·레이아웃·컴포넌트 템플릿
- API 라우트·서비스·도메인 모듈 템플릿
- 문서 템플릿 (ADR, 기능 명세, PR 설명)
- 스킬·규칙·훅 설정 템플릿

## 워크플로

1. `docs/08_PLANS/` 에서 패턴 확정
2. `docs/06_TEMPLATES/` 에 템플릿 추가
3. Build 시 템플릿을 복사·변형하여 `src/` 에 적용
4. 스킬(`docs/05_AI/skills/`)과 연동하여 자동 스캐폴딩

## 원칙

동일한 구조를 매번 수동으로 만들지 않습니다 (Reusable System).
