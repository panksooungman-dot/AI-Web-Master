전환 CTA = 상담신청

1) 전체 전환 스토리 (공통 퍼널)

모든 시나리오가 최종적으로 도달하는 공통 경로입니다.

2) 사용자 시나리오별 화면 스토리보드

■ 네이비 버튼 = 1차 CTA(상담신청) ■ 골드 영역 = 보조 CTA/강조 → 화살표 = 다음 화면으로의 사용자 이동

3) 미팅 확인 포인트

  • 홈 퀵서치 조건(지역·면적·분양가·역세권)이 의뢰자 실제 상품과 맞는가?
  • 리스트 카드에 보여줄 정보(면적/분양가/층/입점/역세권/호재1줄) 충분한가?
  • 지도 마커 클릭 시 프리뷰 → 상세 흐름이 자연스러운가?
  • 비교는 최대 몇 건(3 또는 4)으로 할지?
  • 상담신청 필수 필드(이름·연락처·개인정보동의) 외 추가할 항목이 있는가?
