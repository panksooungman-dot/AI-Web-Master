## 1. 프로젝트 개요

### 1.1 서비스 정의
상가 **분양·매물 탐색 → 상세 확인 → 상담신청**까지를 하나의 여정으로 제공하는 B2C 부동산 웹사이트입니다.

### 1.2 타깃 사용자

| 구분 | 니즈 |
|------|------|
| 투자자 | 역세권·소형·수익형 상가 조건 검색 및 호재 확인 |
| 창업자 | 유동·입지 중심 탐색 후 창업 적합 매물 상담 |
| 자문형 사용자 | 다수 매물 비교 후 상담 요청 |

### 1.3 핵심 기능
- 분양/매물 리스트, 검색·필터
- 상가 상세(호재·입지·평면도)
- 지역·역세권 지도 탐색
- 관심등록, 상가 비교(최대 3~4건)
- 상담신청(리드 수집)
- FAQ, 공지, 고객 후기, 회사소개(신뢰)

### 1.4 CTA 원칙

| 우선순위 | CTA | 역할 |
|----------|-----|------|
| 1차 | **상담신청** | 전환(리드) |
| 2차 | 관심등록 / 비교담기 / 지도에서 보기 / 상세보기 | 탐색·재방문 유도 |

### 1.5 디자인·UX 가이드라인 (합의 가정)
- 톤: 부동산 신뢰형 — 모던 **네이비/화이트**, 포인트 **골드 또는 그린**
- Header: 로고 + 메뉴 + 검색 + 관심/마이페이지
- Footer: 이용약관 / 개인정보처리방침 / 문의처 / 사업자정보
- 모바일: 검색·필터 = 드롭다운·모달·바텀시트 / 상담 = 스티키·바텀시트
- 접근성: 대비, 버튼 라벨 명확, 폼 필수(`*`) 표기

## 3. 핵심 전환 퍼널

```
① 홈          →  퀵서치 / 추천 매물 진입
② 리스트·검색 →  조건 필터로 매물 축약
③ 상가 상세   →  호재·입지·평면도로 확신
④ 상담신청    →  리드 폼 제출
⑤ 완료        →  접수번호·예상 연락 안내
```

| 단계 | 화면 | 성공 지표(예시) |
|------|------|-----------------|
| 1 | P01 | 검색·추천 클릭률 |
| 2 | P02 / P03 | 필터 적용 후 상세 진입률 |
| 3 | P04 | 상담·관심·비교 CTA 클릭률 |
| 4 | P07 | 폼 제출 완료율 |
| 5 | P07 완료 | 리드 유효 연락처 비율 |

## 4. 사용자 시나리오

### 시나리오 1 — 서울 역세권 소형 상가 투자자
1. 홈에서 지역=서울, 면적=소형, 역세권 조건 검색
2. 리스트에서 가격·역세권 정렬 후 후보 확인
3. 상세에서 호재·입지 검토 후 상담신청

**핵심 화면:** P02 리스트 · P04 상세 · P07 상담신청

### 시나리오 2 — 유동인구 많은 상권 창업 희망자
1. 지도에서 상권·유동 구간 탐색
2. 마커 클릭 → 프리뷰 카드로 1차 판정
3. 상세 입지·업종 적합 확인 → 관심 후 상담

**핵심 화면:** P03 지도 · P04 상세 · P07 상담신청

### 시나리오 3 — 다수 매물 비교 후 상담(자문형)
1. 리스트에서 비교담기(3~4건)
2. 비교표로 면적·분양가·입점·역세권·호재 대조
3. 최종 1~2건 상세 재확인 후 「비교상담」으로 상담신청

**핵심 화면:** P05 비교 · P04 상세 · P07 상담신청
