# 오늘 결정한 사항

- CURRENT_CONTEXT.md는 AI가 가장 먼저 읽는 문서로 사용한다.
- Work History 시스템을 구축한다.
- 간단한 수정은 VS Code에서 직접 수정한다.
- 복잡한 수정은 Claude에게 요청한다.
- exit는 Git 명령이 아니라 업무 종료 명령으로 발전시킨다.
- 작업 종료를 `/endday`(문서 정리, Git 없음)와 `exit`(Git commit/push)로 역할 분리한다.
- CURRENT_CONTEXT.md는 덮어쓰기(최신 상태만), WORK_HISTORY.md는 append(이력 보존)로 갱신 방식을 구분한다.
