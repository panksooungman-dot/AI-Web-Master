# docs

프로젝트의 **문서 허브**입니다. AI 에이전트와 사람이 공유하는 맥락, 계획, 구현 기록, 스킬 정의를 한곳에서 관리합니다.

## 하위 디렉터리

| 폴더 | 용도 |
|------|------|
| `context/` | 프로젝트 배경, 용어, 제약, 의사결정 맥락 |
| `plan/` | 설계·아키텍처·기능 명세 (Plan 단계 산출물) |
| `build/` | 구현 가이드, 체크리스트, 빌드·배포 기록 |
| `skills/` | 에이전트 스킬 문서 및 워크플로 정의 |

## 원칙

- **Documentation First**: 구현 전후로 관련 문서를 여기에 반영합니다.
- 루트의 `PROJECT_VISION.md`, `AGENTS.md`와 함께 단일 진실 공급원(SSOT)으로 활용합니다.
- `@docs` 로 Cursor 채팅에 맥락으로 첨부할 수 있습니다.
