# Agent Execution

| Agent | Responsibility |
|--------|----------------|
| Business Analyst | Analyze requirements |
| Product Manager | Define roadmap and priorities |
| Solution Architect | Design system architecture |
| Backend Engineer | Build APIs and business logic |
| Frontend Engineer | Develop user interface |
| AI Engineer | Integrate AI capabilities |
| DevOps Engineer | Configure deployment pipeline |
| QA Engineer | Validate functionality |
| Technical Writer | Produce documentation |

# Best Practices

Always:

- Follow the standard workflow.
- Reuse existing Agents and Prompts.
- Keep Memory updated.
- Validate every deliverable.
- Document major decisions.
- Follow coding standards.

---

# Success Criteria

The project is successful when:

- Requirements are satisfied.
- Architecture follows standards.
- Code passes testing.
- Documentation is complete.
- Deployment succeeds.
- Memory reflects the latest project state.
