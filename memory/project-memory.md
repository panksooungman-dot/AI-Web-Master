# Project Memory

## Overview

Project Memory stores the persistent knowledge of a project throughout its lifecycle.

It provides a single source of truth for project objectives, architecture, progress, milestones, constraints, and historical context.

This memory should evolve as the project evolves.

---

# Purpose

Maintain long-term project context so that AI Agents can make consistent and informed decisions.
