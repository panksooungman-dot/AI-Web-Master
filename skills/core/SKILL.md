# Core

## Overview

`skills/core/` defines the operating principles shared by every other Skill category
(`domains/`, `experts/`, `shared/`, `templates/`, `workflows/`). These are not tied to a
specific role or technology — they describe how any AI Business OS Skill should behave.

## Contents

| Skill | Purpose |
|---|---|
| `ai-business-os-core/` | Baseline operating rules referenced by all Skills |
| `communication/` | How Skills report status, ask questions, and hand off work |
| `decision-making/` | How to choose between implementation options and when to escalate |
| `documentation/` | Documentation conventions (see also `docs/02_DEVELOPMENT/CNBIZ_RULES.md`) |
| `karpathy-guidelines/` | Code quality/simplicity guidelines |
