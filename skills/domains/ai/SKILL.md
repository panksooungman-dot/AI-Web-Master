# AI Domain

## Purpose

This skill defines best practices for designing, developing, and operating AI-native applications.

The objective is to build reliable, secure, scalable, and responsible AI systems that deliver measurable business value through intelligent automation and decision support.

---

# When to Use

Execute when:

- Building AI products
- Integrating LLMs
- Developing AI agents
- Implementing RAG systems
- Creating AI copilots
- Building machine learning applications

---

# Objectives

Build AI systems that are:

- Reliable
- Explainable
- Secure
- Scalable
- Cost-efficient
- Responsible

---

- Business Requirements
- AI Use Cases
- Data Sources

- Model Requirements
- Compliance Requirements

# Model Management

- Model Selection
- Versioning
- Fine-tuning
- Evaluation
- Cost Monitoring
- Latency Monitoring

# Responsible AI

Ensure:

- Human Oversight
- Bias Evaluation
- Privacy Protection
- Safety Controls
- Transparency
- Auditability

Review AI outputs before high-impact decisions.

---

# Evaluation

Measure:

- Accuracy
- Hallucination Rate
- Response Quality
- Latency
- Cost per Request
- User Satisfaction

Continuously evaluate production performance.

---

# Security

Protect:

- API Keys
- User Data
- Model Endpoints
- Prompt Injection Risks
- Data Leakage
- Tool Permissions

Never expose sensitive information.

# Validation Checklist

Before completion verify:

- AI objectives defined
- Knowledge sources validated
- Prompts tested
- Model evaluated
- Safety controls enabled

---

# Failure Conditions

Stop and request clarification if:

- AI objectives are unclear
- Knowledge sources are unavailable
- Model selection is inappropriate
- Safety requirements are undefined
- Evaluation criteria are missing

---

# Rules

- Keep humans in the loop for critical decisions.
- Evaluate models continuously.
- Protect sensitive information.
- Optimize for measurable business value.
- Improve prompts and knowledge iteratively.

---

# Success Criteria

This skill succeeds when:

- AI reliably solves the intended business problem
- response quality meets expectations
- safety and privacy requirements are satisfied
- operational costs remain sustainable
- users trust the AI system
