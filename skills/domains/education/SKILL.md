# Education Domain

## Purpose

This skill defines best practices for designing and implementing education technology platforms.

The objective is to support effective teaching, learning, assessment, and administration through secure, scalable, and learner-centered digital systems.

Education platforms should improve learning outcomes while simplifying educational operations.

---

# When to Use

Execute when:

- Building Learning Management Systems (LMS)
- Developing online education platforms
- Supporting schools, universities, or training organizations
- Creating e-learning or certification systems

---

# Objectives

Build education platforms that are:

- Learner-centered
- Scalable
- Secure
- Accessible
- Data-driven

- Learning Requirements

- Reporting Requirements

# User Management

Support roles such as:

- Student
- Instructor
- Teaching Assistant
- Administrator
- Parent (if applicable)

# Course Management

Manage:

- Courses
- Modules
- Lessons
- Learning Materials
- Assignments
- Schedules

Organize content into structured learning paths.

---

# Learning Experience

Provide:

- Video Lessons
- Reading Materials
- Interactive Activities
- Quizzes
- Discussions
- Progress Tracking

Support self-paced and instructor-led learning.

---

# Assessments

Support:

- Quizzes
- Exams
- Assignments
- Projects
- Peer Reviews
- Automatic Grading

Ensure assessment integrity and fairness.

---

# Progress Tracking

Track:

- Course Completion
- Assessment Scores
- Attendance
- Learning Progress
- Certifications
- Engagement Metrics

Provide timely feedback to learners.

---

# Communication

Enable:

- Announcements
- Messaging
- Discussion Forums
- Notifications
- Calendar Events

Facilitate collaboration between learners and instructors.

# Accessibility

Follow:

- WCAG 2.2 AA
- Responsive Design
- Keyboard Navigation
- Screen Reader Support

# Reporting & Analytics

# Validation Checklist

Before completion verify:

- Assessments validated

# Failure Conditions

Stop and request clarification if:

- Learning objectives are undefined
- Assessment strategy is incomplete
- User roles are unclear
- Accessibility requirements are missing
- Reporting requirements are incomplete

# Rules

- Prioritize learner success.
- Support inclusive education.
- Protect student data.
- Measure learning outcomes.
- Continuously improve the learning experience.

# Success Criteria

This skill succeeds when:

- learners can complete educational programs successfully
- instructors can manage courses efficiently
- assessments are reliable
- learning progress is measurable
